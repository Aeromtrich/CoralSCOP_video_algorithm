# 从零到看懂 CoralSCOP_video 算法的学习路径

## 第一阶段：编程基础

**Python**
- 变量、函数、类、文件读写
- 列表、字典、NumPy 数组操作
- 推荐资源：《Python编程：从入门到实践》

**Linux 基本命令**
- `cd`, `ls`, `python xxx.py`, conda 环境管理
- 能在终端跑起来一个脚本就够了

---

## 第二阶段：数学基础

- **线性代数**：矩阵乘法、向量、维度变换（理解"特征"是什么）
- **概率统计**：均值、方差、概率分布
- **微积分**：导数、梯度（理解"模型怎么学习"）

推荐资源：3Blue1Brown 的 YouTube 系列（有中文字幕，非常直观）

---

## 第三阶段：深度学习入门

- 什么是神经网络、层、激活函数
- 什么是损失函数、反向传播、梯度下降
- **PyTorch 基础**：Tensor 操作、写一个简单的训练循环
- 推荐资源：李沐《动手学深度学习》（有免费中文版和视频）

---

## 第四阶段：计算机视觉基础

- 图像是什么（像素、通道、分辨率）
- **卷积神经网络 CNN**：卷积、池化、特征图
- OpenCV 基础：读图、缩放、画框
- 经典任务概念：分类 / 检测 / 分割 的区别

---

## 第五阶段：Transformer 架构

- **Attention 机制**（注意力机制）是什么
- **Transformer** 的基本结构（Encoder/Decoder）
- **ViT**：把图像切成 patch 当成序列来处理
- 推荐资源：李沐精读论文系列《Attention Is All You Need》

---

## 第六阶段：图像分割专项

- 语义分割 vs 实例分割 vs 全景分割 的区别
- **SAM（Segment Anything Model）** 论文原文阅读
- COCO 数据集格式、RLE 编码是什么

---

## 学习时间参考

| 阶段 | 预估时间（每天2小时） |
|------|------|
| Python 基础 | 1-2 个月 |
| 数学基础 | 1-2 个月 |
| 深度学习入门 | 2-3 个月 |
| 计算机视觉 | 1-2 个月 |
| Transformer | 1 个月 |
| 分割专项 + SAM | 2-4 周 |

**总计大约 6-12 个月**，能达到"看懂并修改这个代码"的水平。

---

## 务实建议

不要追求"全都学完再看代码"。建议：

1. 先学 Python + PyTorch 基础
2. 跑通 `test.py`，让程序能运行起来
3. 带着"这行代码是干什么的"这个问题去查资料
4. **边跑边学**，效率远高于先学完再动手
